It will utilize user input since the user can select the city he/she wants to view. Furthermore, the program will use a list to store the different cities. It will use iteration with a parameter with the current weather data to update and display the weather conditions every few minutes. Based on the type of weather, the program will use selection to display the appropriate image for the weather conditions as well as to display weather alerts. For example, if it is sunny, the background will depict a sunny background.
